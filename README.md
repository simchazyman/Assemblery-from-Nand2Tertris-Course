# Hack Assembler

A C# implementation of the Hack assembler from the Nand2Tetris course.

## Features

- Two-pass assembly
- Symbol table support
- Labels and variables
- Translation of A- and C-instructions
- Generates .hack machine code files

## Structure

- Parser.cs
- Code.cs
- SymbolTable.cs
- Assembler.cs
- Program.cs

## Usage

dotnet run Pong.asm
